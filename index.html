How to play: Use your arrow keys to move the tiles. When two tiles with the same number touch, they merge into one! This version of 2048 includes black question tiles which, when merged, present you with a question. You can't continue playing until you answer it!

Question sets: The default questions are examples. The true power of this game comes from the "Quizlet URL" option in the dropdown. After quickly reviewing a Quizlet flashcard set like http://quizlet.com/14492765/physics-flash-cards/, you only need to copy the link to start studying with 2048!
